This document catalogs all resources in the game, their sources, production chains, and uses. Resources flow from raw materials through various processing stages to create refined products.

## Resource Categories

### Raw Materials
Resources extracted directly from the environment or basic gathering operations.

### Processed Materials
Resources created by processing raw materials through production buildings.

### Refined Products
High-quality resources created through specialized processing and refinement.

### Specialized Resources
Unique resources created through advanced processes or special techniques.

## Resource Production Chains

### Wood & Timber Chain

**Raw → Processed → Refined**

1. **Lumber** (Raw)
   - Source: Timberworks
   - Tier 1: +10/day
   - Tier 2: +25/day
   - Tier 3: +50/day
   - Tier 4: +80/day
   - Tier 5: +120/day
   - Tier 6: +200/day
   - Uses: Construction, crafting, fuel

2. **Refined Lumber** (Processed)
   - Source: Timberworks (Tier 3+)
   - Tier 3: +5/day
   - Tier 4: +8/day
   - Tier 5: +15/day
   - Tier 6: +25/day
   - Created From: Lumber (processed by Timberworks)
   - Uses: Advanced construction, furniture, refined crafting

3. **Spirit Timber** (Refined)
   - Source: Spiritwood Atelier
   - Created From: Lumber + Refined Lumber + Qi Crystals
   - Uses: Spirit-infused construction, formations, advanced crafting

4. **Spirit Planks** (Refined)
   - Source: Timberworks (Tier 6), Spiritwood Atelier
   - Created From: Refined Lumber + Spirit Timber
   - Uses: Elite construction, spirit formations, master crafting

5. **Resonant Timber** (Specialized)
   - Source: Spiritwood Atelier (Tier 5+)
   - Created From: Spirit Timber + advanced processing
   - Uses: Specialized formations, resonance-based crafting

## Resource Flow Summary

### Primary Production Chains

1. **Wood Chain**: Lumber → Refined Lumber → Spirit Timber → Spirit Planks
2. **Stone Chain**: Stone Blocks → Cut Stone → Spirit Stone Fragments → Attuned Stone → Formation-grade Stone
3. **Metal Chain**: Iron Ore → Iron Ingots → Steel Ingots → Spirit Alloys → Celestial Alloy
4. **Essence Chain**: Essence Pebbles → Essence Shard → Essence Stones → Spirit Core → Celestial Cores
5. **Qi Chain**: Qi Crystals → Mana Crystals → Spirit Crystals → Qi Cores
6. **Food Chain**: Food → Processed Food → Spirit Infused Meals
7. **Herb Chain**: Raw Herbs → Refined Herbs → Herb Powder → Elixirs → Rare Elixirs
8. **Water Chain**: Water Tokens → Purified Water → Distilled Water → Spirit Dew → Mist Crystals

### Cross-Chain Dependencies

- **Spirit-infused resources** require both base material and qi/essence resources
- **Formations** require stone + essence + qi resources
- **Advanced crafting** typically requires multiple resource chains combined
- **High-tier buildings** often require resources from multiple chains

## Notes

- Resource production rates are per building instance per day
- Multiple buildings can produce the same resource for increased throughput
- Resources flow from raw to processed to refined, with some specialized endpoints
- Supply chains and proximity bonuses can affect production rates
- Some resources are byproducts of other processes (e.g., Fertilizer from food production)
